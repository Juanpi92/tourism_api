API Documentation

O que essa api faz?

Essa é uma API que fornece informações sobre alguns passeios turísticos no estado do Rio de Janeiro, Brasil. Cada passeio tem um ID, nome, preço, descrição, localização, coordenadas, lista de itens inclusos, capacidade máxima, quantidade de vendas e imagens. Além disso, há uma classificação por região e por avaliação do passeio. Os passeios incluem um passeio de escuna pelas ilhas de Angra dos Reis, um passeio completo em Cabo Frio com passeio de lancha e churrasco à bordo, um passeio de escuna em Ilha Grande, uma visita ao distrito de Itaipava com visita ao Castelo e ao Parque Municipal, uma visita a Paraty e uma trilha até a Pedra do Telégrafo. As informações são fornecidas em formato JSON.

URL da API de consulta

Follows the API URL pattern of Azul Tour

  • URL pattern: https://tourismapi.herokuapp.com/products/[id]
  • Exemple: https://tourismapi.herokuapp.com/products/5

POST /products

Inserts a new product in the database.

Returns a JSON object with the property insert defined as true.

Body

Property Type description
name string Product's name
price number price of the product
description string description do product
located string Where is the product located
included Array of strings product capacity
capacity Number List of items included in the product
sold Number Quantity of products sold
images Array of strings Product photo list
region String region where the product is located
Latitude String Provides geolocation in latitude
longitude String Provides geolocation in longitude
rating number Provides product review
date string Provides the available date for the product
Type Response
Json { 'insert'= true }

DELETE /products/:id

Removes a specific product, identified by the parameterid, from the database.

Returns a JSON object with the property delete defined as true.

Response

database.
name Type description
id integer Product ID to be removed
Type Response
Json { "Delete":true }

Parameters

included Array of strings product capacity

sold Number Quantity of products sold
images Array of strings Product photo list
region String region where the product is located
Latitude String Provides geolocation in latitude
longitude String Provides geolocation in longitude
rating number Provides product review
date string Provides the available date for the product

Response

Type Response
Json { 'update':true }

Returns a JSON object with the property update defined as true.

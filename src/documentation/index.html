URL da API de consulta

Segue o padrão da URL da API de consulta para buscar o endereço, cidade e estado através do CEP.

  • Padrão da URL: https://tourismapi.herokuapp.com/products/[id]
  • Exemplo: https://tourismapi.herokuapp.com/products/5

GET /products

Retorna uma lista de todos os produtos cadastrados na base de dados.

Resposta

Propriedade Tipo Descrição
id inteiro ID do produto
nome string Nome do produto
descricao string Descrição do produto
preco número Preço do produto
imagens array de strings Lista de imagens do produto
included array de strings Lista de itens inclusos com o produto

GET /products/:id

Retorna os detalhes de um produto específico, identificado pelo parâmetro id.

Parâmetros

Nome Tipo Descrição
id inteiro ID do produto a ser recuperado

Resposta

Propriedade Tipo Descrição
id inteiro ID do produto
nome string Nome do produto
descricao string Descrição do produto
preco número Preço do produto
imagens array de strings Lista de imagens do produto
included array de strings Lista de itens inclusos com o produto

POST /products

Insere um novo produto na base de dados.

Retorna um objeto JSON com a propriedade insert definida como true.

Resposta

Propriedade Tipo Descrição
nome string Nome do produto
descricao string Descrição do produto
preco número Preço do produto
imagens array de strings Lista de imagens do produto
included array de strings Lista de itens inclusos com o produto

DELETE /products/:id

Remove um produto específico, identificado pelo parâmetro id, da base de dados.

Retorna um objeto JSON com a propriedade delete definida como true.

Resposta

Nome Tipo Descrição
id inteiro ID do produto a ser removido

PUT /products/:id

Atualiza um produto específico, identificado pelo parâmetro id, na base de dados.

Parâmetros

Nome Tipo Descrição
id inteiro ID do produto a ser atualizado

Corpo da Requisição

Propriedade Tipo Descrição
nome string Nome atualizado do produto
descricao string Descrição atualizada do produto
preco número Preço atualizado do produto
imagens array de strings Lista de imagens atualizada do produto
included array de strings Lista de itens inclusos atualizada do produto

Resposta

Retorna um objeto JSON com a propriedade update definida como true.

O que é um token JWT?

Um token JWT, ou JSON Web Token, é um padrão aberto (RFC 7519) que define um formato compacto e autônomo para transmitir informações de forma segura entre partes como um objeto JSON assinado digitalmente.

Em geral, um token JWT é usado como uma forma de autenticação e compartilhamento de informações entre diferentes sistemas e aplicativos. Ele é frequentemente usado em aplicações web e móveis para verificar a identidade do usuário e autorizar o acesso a recursos ou dados específicos.

Items needed for user registration

Property Type Description
name string username
email string user email
password string user password
idioms string languages ​​the user knows
profession string user's profession
located string user location
tel1 string first user phone
tel2 string second user phone
about string about the user
images array of string profile pictures
image_banner string profile background photo
image_profile string user photo

Registry response examples

Property Type Description Json
error string Error message indicating that the email is already registered 
  {
    "error": "User already exist"
  }
register boolean Indicates whether the registration was successful 
  {
    "register": true
  }
